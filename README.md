Rode com um sistema Linux, WSL ou em outro SO/Sistema que possa rodar o Makefile, para assim rodar tudo usando o comando **make**. Se preferir compile manualmente e execute ./mutualExclusion **num_threads** (o que irá rodar com 5 contas, que é o padrão), onde **num_threads** é o número de threads que desejar que o programa execute.

Após isso serão gerados 5 arquivos log para os extratos finais da conta bancária, além da impressão no terminal dos saldos finais. Use **make clean** para limpar o mutualExclusion compilado e os logs ou faça isso manualmente.
